### Build
Open **Trysil.Expert[Version].dproj** in this directory and Build the project in Release.<br><br>
Alternatively, run **Build[Version].bat** from a command prompt &mdash; it compiles **Resources\Trysil.Expert.Resources.rc** with brcc32 and then builds the matching project in Release for the platforms shipped with that Delphi version (Win32 only on 260/270/280, Win32 and Win64 on 290/370).
